# Context
Memory acquisition has became a standard in the DFIR industry.
Before, it was considered useless because the RAM was small and hard to collect.
Now, it contains gigabytes of data, lots of artifacts and is way easier to collect thanks to tools like [[avml]], ...

# Use cases
In the memory, you can find : 
- ransomwares encrypting keys
- clear text passwords
- fileless malwares
- DLL injections and process hollowing
- Private navigation of web browsers
- Clipboard artifacts (copy/paste)
