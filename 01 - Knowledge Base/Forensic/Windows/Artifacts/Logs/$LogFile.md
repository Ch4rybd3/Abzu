# Context
The [[$LogFile]] is the log for every actions that happens on an [[NTFS]] system.
Before updating the [[$MFT]], file changes are logged in the `$LogFile`.
It's main purpose is for integrity. If the machine is shut down, the [[$LogFile]] is used to repair the [[$MFT]]
